# Matignon - Mon Suivi : Idées de Design

## Contexte
Application mobile (affichée en format web) de suivi client pour la Clinique Matignon.
Identité de la marque : fond noir, typographie majuscule bold, vert fluo (#B8FF00) comme accent, style médical haut de gamme.

## Approche 1 — Clinique Luxe Sombre (Dark Medical Luxury)

**Design Movement** : Médical Premium / Dark Luxury UI

**Core Principles** :
- Fond noir profond avec des accents vert fluo signature Matignon
- Typographie monumentale en majuscules pour les titres
- Cartes avec bordures fines lumineuses sur fond sombre
- Hiérarchie visuelle forte : ce qui est important ressort sur le noir

**Color Philosophy** :
- Background : #0A0A0A (noir profond)
- Accent : #B8FF00 (vert fluo Matignon)
- Surface : #141414 / #1C1C1C
- Texte : #FFFFFF / #A0A0A0 (secondaire)
- Danger/Alerte : #FF4444

**Layout Paradigm** :
- Format mobile centré (max-width 390px) avec fond noir global
- Navigation bottom bar avec icônes minimalistes
- Cards avec left-border accent vert fluo
- Timeline verticale pour l'historique des traitements

**Signature Elements** :
- Barre de progression circulaire pour "prochain rendez-vous"
- Badge vert fluo pour les recommandations urgentes
- Ligne horizontale fine vert fluo comme séparateur de section

**Interaction Philosophy** :
- Tap sur une carte → expand avec animation slide-up
- Swipe pour naviguer entre les onglets
- Feedback haptique simulé via micro-animations

**Animation** :
- Entrée des cartes : fade + translateY(20px) → 0
- Progress bars : fill animation au chargement
- Transitions de page : slide horizontal

**Typography System** :
- Titres : Bebas Neue (majuscule, condensé, bold)
- Corps : DM Sans (lisible, moderne)
- Labels : DM Mono (monospace pour les données médicales)

## Approche 2 — Wellness Épuré Crème (Organic Wellness)

**Design Movement** : Organic Minimalism / Spa Premium

**Core Principles** :
- Fond crème/beige chaud avec typographie serif élégante
- Espace blanc généreux, respiration visuelle
- Tons naturels évoquant la peau, le soin, la douceur
- Contrastes doux mais lisibles

**Color Philosophy** :
- Background : #FAF7F2 (crème chaud)
- Accent : #2D2D2D (noir doux) + #C8A882 (or rosé)
- Surface : #FFFFFF / #F0EBE3
- Texte : #1A1A1A / #7A6E65

**Layout Paradigm** :
- Format mobile centré avec marges généreuses
- Sections séparées par des espaces blancs larges
- Cards avec coins légèrement arrondis et ombre douce
- Illustrations linéaires minimalistes

**Signature Elements** :
- Icônes en trait fin style "medical illustration"
- Séparateurs ondulés organiques
- Avatar client avec initiales en cercle doré

**Interaction Philosophy** :
- Interactions douces et fluides, jamais brusques
- Hover states avec légère élévation de carte

**Animation** :
- Fade in progressif des sections
- Micro-animations sur les icônes au survol

**Typography System** :
- Titres : Playfair Display (serif élégant)
- Corps : Lato (neutre, lisible)

## Approche 3 — Techno Médical Asymétrique (Medical Tech Asymmetric)

**Design Movement** : Medical Tech / Brutalist Clean

**Core Principles** :
- Fond blanc cassé avec accents noirs tranchants et vert fluo Matignon
- Mise en page asymétrique avec des blocs décalés
- Typographie condensée et bold pour un impact visuel fort
- Données médicales affichées comme un dashboard tech

**Color Philosophy** :
- Background : #F5F5F0 (blanc cassé légèrement chaud)
- Accent primaire : #0A0A0A (noir Matignon)
- Accent secondaire : #B8FF00 (vert fluo)
- Surface : #FFFFFF
- Texte : #0A0A0A / #666666

**Layout Paradigm** :
- Format mobile centré (max-width 390px)
- Header asymétrique avec logo à gauche, menu hamburger à droite
- Blocs de contenu décalés horizontalement (pas tous centrés)
- Timeline horizontale scrollable pour les traitements passés

**Signature Elements** :
- Numéros de section en très grand (style editorial magazine)
- Bordure gauche épaisse noire sur les cartes importantes
- Tags rectangulaires sans arrondi pour les catégories de soins

**Interaction Philosophy** :
- Interactions précises et directes
- Animations rapides et nettes (pas de flou)

**Animation** :
- Slide-in depuis la gauche pour les nouvelles sections
- Scale légère sur les boutons au clic

**Typography System** :
- Titres : Barlow Condensed (condensé, impactant)
- Corps : IBM Plex Sans (technique, lisible)
- Données : IBM Plex Mono

## Choix retenu : Approche 1 — Clinique Luxe Sombre (Dark Medical Luxury)

Cette approche correspond le mieux à l'identité visuelle de Matignon : fond noir, vert fluo, typographie bold en majuscules. Elle crée une expérience premium et cohérente avec le site existant de la clinique.
